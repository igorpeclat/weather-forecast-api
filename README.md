# weather-forecast-api
RESTFull Api for average the waether forecast

## Pre-requisites
Java 8+

## How-to Build

### Initialize weather-forecast-api
```mvn clean install```

### Test
```mvn test```

### How-to Run
```mvn spring-boot:run```
